### Pré-requisitos
- Python 3.10+
- Node.js 20.x
- PostgreSQL

### Passos para Instalação (Exemplo)
1. Clone o repositório: `git clone ...`
2. Instale as dependências do backend: `pip install -r requirements.txt`
3. Instale as dependências do frontend: `npm install`
